# SQL RFM Sales Analysis

## Overview

This project analyzes retail sales data using SQL and performs **RFM (Recency, Frequency, Monetary)** customer segmentation to understand customer behavior and generate business insights.

## Tools Used

- MySQL
- MySQL Workbench
- SQL

## Dataset

The dataset contains retail sales information, including customer details, products, orders, sales, and order dates.

## What I Analyzed

- Sales by product line
- Sales by year and month
- Revenue by country
- Best customers using RFM analysis
- Products frequently sold together
- Customer segmentation based on RFM scores

## SQL Concepts Used

- SELECT, WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE WHEN
- CTEs (Common Table Expressions)
- Window Functions
- Subqueries
- Date Functions

## Key Learnings

- Performed sales analysis using SQL
- Applied RFM analysis for customer segmentation
- Used CTEs and Window Functions to solve business problems
- Generated business insights from retail sales data
